This library is for the DAC 4725 digital-to-analog converter. It allows you to adjust the output voltage by controlling an Arduino.
1) At first you need to install library
2)Include it to your project
3)Connect DAC4725 device at your arduino board
4)All functions description you can find at .h file
5)Meassure the reference voltage(supply votage)
6)Device is ready to work
